Security & Authentication

🔐 Authentication Flow

  • Sign Up: Email + Password → bcrypt hashing (10 rounds)
  • Login: Email + Password → JWT token (7-day expiry)
  • Storage: MongoDB User model with encrypted passwords
  • Requests: Bearer token in Authorization header

✅ All user data persisted in MongoDB - no ephemeral storage
